Privacy Policy

Effective Date: 01 January 2026

About LeadHive

LeadHive is a browser extension that allows users to manually save information visible on webpages they choose to open. The extension functions only after the user clicks the extension button.

LeadHive is an independent productivity tool and is not affiliated with, endorsed by, or connected to any website or service.

Information Accessed

The extension processes only information that is visible on the currently open webpage and explicitly saved by the user. This may include publicly visible business contact details such as:

  • Business name
  • Business address
  • Phone number
  • Website URL

No hidden data is accessed and no background monitoring occurs.

Data Storage

All saved information is stored locally within the user’s browser using browser storage. The data never leaves the user’s device unless the user exports it manually.

Data Usage

Stored information is used only so the user can review and export saved records for personal organization purposes.

LeadHive does not track users, collect analytics, or profile behavior.

Data Sharing

LeadHive does not sell, transfer, or share any data with third parties. No data is transmitted to external servers.

Data Retention

All saved data remains under the user’s control and can be deleted at any time by clearing extension storage or removing the extension.

Permissions Usage

  • activeTab – used only after user interaction to read visible page content
  • storage – used to store saved records locally
  • scripting – used to access page content after user activation
